How to use this web app

  1. Open the address template.
  2. Select if the customer is “Consumer” or “Commercial/Distribution”.
  3. 3. Fill the required customer information:
    • Company (Only for Commercial/Distribution)
    • Contact (POC)
    • Email:
    • Telephone number:
      • If you see a green check mark ( ) it means the information has been verified, if not, you will receive a warning message ( ), so please double check the information.
  4. Put the address you want to validate in front of “Search your address:”.
    The app will automatically verify the data and will show all the details, including the exact location in Google Maps.
  5. Now you will see all the information you entered/verified below “Ship to:”.
    Note that all components of the information must have a green check mark on the right side, a red exclamation mark means there is still an error.
  6. If you want to copy the organized information in your clipboard, you can do so by clicking the “Copy information” button.
