How to use this web app

  1. Open the address form.
  2. Fill the customer information that is asked and scroll down.
  3. Fill the address information then click on validate. Now you will see a quick comparison between the address you entered and the one the systems verifies. If you see this fields written in green it means you address is verified and correct, if not, please double check the information.
  4. Close the address form by clicking “Return Home” on the top right corner.
  5. Now you will see all the information you entered/verified below “Ship to:”. Note that all parts of the address must have a green check mark on the right side, a red exclamation mark means there is still an error.
  6. If you want to copy the organized information, you can do so by clicking the “Copy information” button.
